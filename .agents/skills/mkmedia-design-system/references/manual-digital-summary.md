# MK MEDIA Manual 2025: Digital Summary

Source: `MANUAL DE MARCA MKMEDIA 2025.pdf` (created January 10, 2026).

## Core Identity

- Brand naming in manual: `MK MEDIA`
- Visual direction: strategic, professional, human, and results-oriented.

## Logo Rules

- Use official MK MEDIA logo lockup.
- Minimum size: `20px`.
- On white/light backgrounds: prioritize blue logo version.
- On dark/solid brand backgrounds: use white logo version.
- Keep clear space around the logo (do not crowd with nearby UI elements).

## Official Palette

- Blue: `#0359A8`
- Yellow: `#FCB814`

Recommended digital semantics:
- Blue for identity, navigation emphasis, active states, and primary accents.
- Yellow for counters, highlights, and high-attention accents.

## Typography

- Primary typeface: `Myriad Pro`.
- For web implementation:
  - Use licensed local font files when available.
  - Keep a resilient fallback stack for environments without the licensed files.

## Practical UI Guidance

- Keep base surfaces neutral to maintain legibility and hierarchy.
- Apply brand colors with intent, not as full-page tinting by default.
- Maintain strong contrast for text/icons over brand colors.
- Ensure responsive consistency: same logo logic and token usage on desktop and mobile.
